# 🚀 QuestPlanner

AI-планировщик задач с Telegram-ботом и DevOps инфраструктурой.

---

## ⚡ Features

- 🤖 AI ответы (OpenAI)
- 📱 Telegram бот
- ⚙️ FastAPI backend
- 🐳 Docker (multi-container)
- 🔁 Jenkins CI/CD
- 📊 Monitoring (Prometheus + Grafana)
- ☁️ Terraform (IaC)
